## Notes on scope explicitly excluded by the audit brief

- **Counterparty-facing Governance Record** — not in Phase 1; no UI exists, no claim made.
- **PDF evidence pack / external export / SIEM export** — not in Phase 1; no UI exists, no claim made.
- **Raw provider payload viewer** — explicitly forbidden; `redactMetadata` strips secrets and oversized strings. Drawer shows only redacted JSON.
- **Reason-code/taxonomy hard BLOCK enforcement** — not in Phase 1; reason codes are surfaced but not gating in the UI.
- **Full waiver enforcement into progression gates** — out of scope; waivers are visible and granted via panel, enforcement remains backend hooks `assertWaiverActive` / `consumeGovernanceWaiver`.
- **Payment webhook atomicity** — out of scope per brief; payment events visible in timeline only.
- **Basic Memory Record** — explicitly out of scope; "Memory record" field intentionally renders "Not wired in this build" with HQ tooltip (`GovernanceRecordDetail.tsx:444-457`).
